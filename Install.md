### INSTALL INSTRUCTION

To install Enterprise commercial professional, you need to import the key file first in Windows 10/11 to enable the install feature. Follow the steps:

**Import the key**
1. Double click the provided key (.pfx).
2. In the Certificate Import Wizard, select Local Machine in the Store Location group box. Click Next.
3. Verify that the File name and path is correct. Click Next.
4. Don't type any password nor check any Import options. Click Next.
5. In the Certificate Store window, select Place all certificates in the following store. Click Browse.
6. Select Trusted Root Certification Authorities in the Select Certificate Store window. Click Ok. Click Next.
7. Click Finish. Click Ok when you see the import is successful.

**Install Enterprise commercial**
1. Double click the installer file (.msix).
2. Click Install button.
